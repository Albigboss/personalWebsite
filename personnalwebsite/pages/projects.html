Projets

Durant mes études, j'ai eu l'opportunité de travailler sur plusieurs projets intéressants qui m'ont permis de développer mes compétences en programmation et en gestion de projet.
Voici une liste de mes projets les plus importants :

Baldus Adventure :

Technologies :

  • C++ avec Qt Creator
  • git
  • Doxygen
  • Windows, Mac, Linux

Baldus Adventure est un jeu vidéo développé en C++ avec Qt Creator. Il est compatible avec Windows, Mac et Linux. Ce jeu a été développé lors d'un projet de fin d'année dans une équipe de 4 personnes.
L'objectif était de créer un jeu "Rétro Gaming" en 2D avec une histoire captivante et des mécaniques de jeu intéressantes.
Durant ce projet, j'ai pu mettre en application mes compétences acquises en C++ mais également en gestion de projet, travail en équipe et communication.
Le déroulement du projet a été divisé en deux grandes parties. Dans un premier temps, la conception du jeu en accord avec le client fictif, puis dans un second temps, le développement du jeu en lui-même avec plusieurs itérations.
J'ai également eu l'occasion de travailler sur le versionning du code avec git.

Création d'un serveur Web :

Technologies :

  • Commandes shell
  • Raspberry Pi
  • HTML/CSS
  • NGINX

La création d'un serveur NGINX a été un projet passionnant qui m'a permis de me familiariser avec la configuration des pages d'erreurs (401, 403, 404) ainsi que des logs du serveur.
J'ai utilisé un Raspberry Pi comme serveur, avec l'os Raspberry Pi OS Lite, ce qui m'a permis d'apprendre à travailler avec du matériel limité. Ce projet m'a également donné l'occasion de renforcer mes compétences en HTML et CSS en créant une interface web simple pour le serveur.
Les serveurs étant en permanence à l'IUT, j'ai dû utiliser le protocole SSH pour me connecter à distance et travailler sur le projet depuis chez moi.

Création d'une base de données SQL pour des hôtels :

Technologies :

  • SQL avec Oracle Database
  • Python
  • Jupyter Notebook

Ce projet de base de données SQL pour des hôtels m'a permis de développer mes compétences en conception et gestion de bases de données relationnelles.
J'ai dû dans un premier temps, avec mon équipe, élaborer la conception d'une base de données en respectant le cahier des charges d'un client fictif.
Nous avons ensuite implémenté cette base de données grâce à des fichiers SQL puis nous avons également créé des utilitaires et une documentation permettant de comprendre le fonctionnement de notre base de données.
Dans un second temps, nous avons reçu un modèle de cette même base de données, mais créé par les professeurs et nous avons dû créer des requêtes SQL en python pour extraire et analyser des données

Hnefatafl :

Technologies :

  • C++ avec Qt Creator
  • Windows, Mac, Linux

Le hnefatafl est un des premiers projets que j'ai réalisé en C++ avec Qt Creator. Il s'agit d'un jeu de stratégie viking où un joueur contrôle les défenseurs et l'autre les attaquants.
Ce projet m'a permis de découvrir les bases de la programmation orientée objet.
La particularité est que ce jeu se jouait dans le terminal.
Pour ce projet, la structure du code nous a été fournie, mais ça été à moi d'implémenter le code dans un premier temps, puis de l'optimiser ensuite.

Recueil de besoins :

Technologies :

  • HTML/CSS
  • Miro
  • Figma

Ce projet de recueil de besoins a été un projet très intéressant qui m'a permis de découvrir les aspects de la création de produit pour un client.
En effet, avec mon équipe, nous avons dû dans un premier temps, interviewer un client pour définir un cahier des charges le plus précis possible. Par la suite, nous avons défini une vision produit ainsi que des personas/scénarios utilisateurs pour avoir la conception la plus optimisée possible. Enfin, nous sommes passés à la mise en place du site web qui est resté un site web statique en HTML/CSS sans mise en production réelle.
